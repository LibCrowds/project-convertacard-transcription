This page contains instructions on how to transcribe the information found on each card.

General Guidance

Filling in the form

  • All fields should be filled in unless there is no relevant information (e.g. no subtitle).
  • Do not add punctuation at the end of any fields unless advised otherwise.
  • Abbreviated guidance for each field can be seen by hovering your cursor over the icon.

Non-Roman scripts

When transliterating non-Roman scripts refer to the Library of Congress Romanization tables:
  • ALA-LC Romanization Tables
  • Non-Slavic Languages (in Cyrillic Script)

Detailed guidance

The following section contains for detailed guidance for each field.

Shelfmark

If the card has a shelfmark transcribe it exactly as given, including upper and lower-case letters and any punctuation and spaces.
