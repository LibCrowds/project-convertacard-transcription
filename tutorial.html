General guidance

When filling in the form:

  • All fields should be filled in unless there is no relevant information (e.g. no subtitle).
  • Rules for entering data in each field vary: in some cases you must transcribe information exactly as given, in others you will record information according to a prescribed format.
  • Do not add punctuation at the end of any fields unless advised otherwise.
  • Abbreviated guidance for each field can be seen by hovering your cursor over the icon.
  • The section headings below match those visible on the form. The same rules apply where the form contains a field for both the romanised form and the original script.
  • Leave any original script fields blank if the original script is Roman.

Non-Roman scripts

Inclusion of information in the original script is strongly recommended. When transliterating non-Roman scripts refer to the Library of Congress Romanization Tables:

  • ALA-LC Romanization Tables
  • Non-Slavic Languages (in Cyrillic Script)

Shelfmark

If the card has a shelfmark transcribe it exactly as given, including upper and lower-case letters and any punctuation and spaces.

ISBN

Record the ISBN if present, omitting any hyphens or spaces, e.g. 0552675873, not 0-552-67587-3.

If there is more than one ISBN, enter the one that identifies the item in hand: e.g. if the hardback and paperback ISBNs are provided, if ISBNs for multiple volume numbers are provided.

Language

Enter the main language of the resource in the Language field (the assoicated language code will be used when the form is submitted).

Additional languages

If there is more than one language, click the checkbox indicating that more than one language is present and enter all additional languages in the Additional Languages field (the assoicated language codes will be used when the form is submitted).

Language note

If the item contains text in more than one language, enter a note on languages used in the resource in the Language Note field.

Examples:

  • In Hungarian; summaries in English and French
  • Includes abstracts in English and Russian
  • Parallel Chinese text and English translation

Script note

Add a note about the original script(s) if this would be important to the user.

Examples:

  • Cyrillic and Arabic script
  • Manchu script

Title

Transcribe the title, capitalising only the first letter and any proper nouns, e.g. A child's Christmas in Wales (rules vary according to language conventions).

Enter any descriptive information which is clearly not on the item itself in the Notes field.

Include only the principal title as given on the title page, adding subsequent text in the section as indicated below:

  • For a remainder of the title (i.e. Madagascar wildlife : a visitor's guide) see Subtitle
  • For a parallel title (i.e. Historical review on ancient dams = Revue historique des anciens barrages) see Parallel title
  • For a statement of responsibility (i.e. Bleak house by Charles Dickens) see Statement of responsibility

Non-filing characters

If the title begins with a symbol or an initial article enter the number of characters plus one for any following space.

Examples:

  • A passage to India = 2
  • Les miserables = 4

Statement of responsibility

Transcribe the statement as given, including any text such as "written by..." or "edited by..."

Subtitle

If there is a subtitle, transcribe it in this column.

Example:

  • Twelfth night : a user's guide

But note that alternative titles, e.g. Twelfth night, or What you will all belong in the main title.

Parallel title

Transcribe a parallel title in this column. If there is more than one, add subsequent titles separated by " = ".

Example:

  • Jahrbuch der Europäischen Gesellschaft für theologische Forschung von Frauen
  • = Yearbook of the European Society of Women in Theological Research
  • = Annuaire de l'Association europ enne des femmes pour la recherche th ologique
